# Media Pack: Module 01

Сюда складываем визуальные материалы для `Модуль 1. Строповка грузов`.

## Папки

- `images` - фото грузов, стропов, ошибок и ситуаций.
- `diagrams` - схемы строповки, углы, чек-листы, логические схемы.
- `audio` - озвучка, дикторские версии и утвержденные звуковые файлы по слайдам.
- `video` - ролики и короткие фрагменты, если появятся.
- `references` - ссылки, исходники, служебные описания медиа.

## Первый приоритет

- типы грузов;
- виды стропов;
- браковка стропов;
- типовые схемы строповки;
- ошибки при обвязке и зацепке.
