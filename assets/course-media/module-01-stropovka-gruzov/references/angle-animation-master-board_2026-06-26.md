# Единая монтажная карта ролика

## Назначение

Этот файл собирает в одном месте всю последовательность ролика:

- порядок углов;
- характер движения;
- смысл каждой остановки;
- визуальные признаки каждого стоп-кадра.

Это рабочая карта для сборки аниматика, чистовой графики и озвучки.

## Последовательность

`30° -> 45° -> 60° -> 90° -> 100°`

После стоп-кадра `100°` добавляется короткий финальный пост-эпизод:

`пауза 2-3 сек -> вибрация стропа -> разрыв с хлопком`

## Общая логика

- крюк неподвижен;
- груз поднимается строго по одной вертикальной оси;
- груз все время параллелен земле;
- видимая длина ветвей уменьшается;
- внутренний участок стропа уплотняется;
- строп наматывается спиралью на внутреннюю часть груза;
- внутреннее расстояние между ветвями сокращается;
- перед каждым стопом после `30°` есть мягкое замедление;
- каждая остановка почти без инерции.

## Монтажная таблица

### `30°`

- роль в ролике: стартовая безопасная точка
- высота груза: самая низкая
- длина ветвей: максимальная
- намотка: слабая
- внутренняя зона: максимально открытая
- визуальный акцент: зеленый
- смысл: потеря `0%`, строп работает на `100%` от номинала

### `45°`

- роль в ролике: первое заметное изменение
- высота груза: немного выше
- длина ветвей: чуть меньше
- намотка: слабая, но уже читаемая
- внутренняя зона: немного уже
- визуальный акцент: желто-зеленый
- смысл: потеря `10%`, допустимая нагрузка `90%`

### `60°`

- роль в ролике: переходная напряженная точка
- высота груза: умеренно выше
- длина ветвей: заметно меньше
- намотка: заметная
- внутренняя зона: ощутимо уже
- визуальный акцент: желто-оранжевый
- смысл: потеря `15%`, допустимая нагрузка `85%`

### `90°`

- роль в ролике: главный объясняющий стоп
- высота груза: заметно выше
- длина ветвей: явно короче
- намотка: явная
- внутренняя зона: узкая
- визуальный акцент: красно-оранжевый
- смысл: потеря `30%`, допустимая нагрузка `70%`

### `100°`

- роль в ролике: финальный предупреждающий стоп
- высота груза: финальная
- длина ветвей: минимальная
- намотка: максимальная
- внутренняя зона: минимальная
- визуальный акцент: красный
- смысл: потеря `40%`, допустимая нагрузка `60%`, критично

### Пост-финал после `100°`

- роль в ролике: финальное предупреждение через наглядное последствие
- задержка: `2-3 секунды`
- состояние груза: еще держится, без перекоса, как будто система на пределе
- поведение стропа: сначала короткая вибрация, затем разрыв
- звук: резкий хлопок
- смысл: критический угол приводит не просто к потере запаса, а к реальному риску разрушения стропа

## Логика переходов

### `30° -> 45°`

- движение мягкое;
- изменения обучающие, без тревоги;
- зритель замечает первый рост нагрузки.

### `45° -> 60°`

- напряжение нарастает;
- намотка и сужение внутренней зоны становятся заметными;
- схема перестает выглядеть нейтральной.

### `60° -> 90°`

- это главный перелом;
- ухудшение должно чувствоваться особенно сильно;
- перед стопом замедление чуть заметнее, чем раньше.

### `90° -> 100°`

- финальное усиление;
- кадр не должен стать тесным;
- смысл не в драме, а в сильном предупреждении.

### `100° -> разрыв`

- после объясняющей фразы кадр удерживается еще `2-3 секунды`;
- в это время строп начинает мелко вибрировать;
- вибрация быстро нарастает, но груз еще остается в геометрии кадра;
- затем происходит резкий разрыв с хлопком;
- сам разрыв показываем коротко и ясно, без натурализма и без лишней зрелищности.

## Что нельзя сломать при сборке

- нельзя смещать груз с центральной вертикали;
- нельзя допускать перекоса груза;
- нельзя двигать точки контакта;
- нельзя терять читаемость угла;
- нельзя делать финал слишком тесным;
- нельзя превращать микроэффект натяжения в грубую деформацию.
- нельзя превращать разрыв в длинную аварийную сцену: это короткий учебный акцент, а не катастрофический эпизод.
