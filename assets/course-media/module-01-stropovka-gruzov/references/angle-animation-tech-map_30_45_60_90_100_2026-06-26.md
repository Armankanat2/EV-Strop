# Техкарта анимации по стоп-кадрам

## Назначение

Эта техкарта фиксирует, как именно меняются ключевые параметры сцены на стоп-кадрах:

- высота груза;
- видимая длина ветвей;
- внутренняя намотка стропа;
- внутреннее расстояние между ветвями;
- визуальное напряжение;
- характер остановки.

Все значения здесь относительные, не расчетные. Это карта для визуальной постановки ролика.

## Общие правила для всех кадров

- крюк неподвижен;
- верхняя точка подвеса неподвижна;
- центр груза движется строго по одной вертикальной оси;
- груз все время параллелен земле;
- перекосов и качания нет;
- точки контакта на грузе не плавают;
- перед каждым стоп-кадром, кроме стартового `30°`, есть короткое плавное замедление;
- в стопе груз замирает почти без инерции.

## Стоп-кадр `30°`

- высота груза: стартовая, самая низкая в последовательности
- видимая длина ветвей: максимальная
- внутренний участок стропа: почти свободный, без выраженного уплотнения
- внутренняя намотка: слабая, только стартовый заход
- внутреннее расстояние между ветвями: максимально открытое
- визуальное напряжение: минимальное
- акцент кадра: безопасное исходное положение
- ветви по цвету/толщине: спокойные, без заметного усиления
- смысл: потеря `0%`, допустимая нагрузка `100%`

## Стоп-кадр `45°`

- высота груза: немного выше стартового уровня
- видимая длина ветвей: чуть меньше, чем на `30°`
- внутренний участок стропа: слегка уплотнился
- внутренняя намотка: слабая, но уже читаемая
- внутреннее расстояние между ветвями: немного уже
- визуальное напряжение: низкое, но уже заметное
- акцент кадра: рост нагрузки начинается
- ветви по цвету/толщине: едва усилены
- характер остановки: мягкое торможение и чистый стоп
- смысл: потеря `10%`, допустимая нагрузка `90%`

## Стоп-кадр `60°`

- высота груза: умеренно выше
- видимая длина ветвей: заметно короче, чем на `30°`
- внутренний участок стропа: уплотнение уже хорошо видно
- внутренняя намотка: заметная
- внутреннее расстояние между ветвями: сократилось ощутимо, но еще не критично
- визуальное напряжение: среднее
- акцент кадра: переходная напряженная точка
- ветви по цвету/толщине: усиление уже читается
- характер остановки: мягкое замедление без рывка
- смысл: потеря `15%`, допустимая нагрузка `85%`

## Стоп-кадр `90°`

- высота груза: заметно выше среднего уровня
- видимая длина ветвей: явно короче
- внутренний участок стропа: плотный
- внутренняя намотка: явная
- внутреннее расстояние между ветвями: узкое
- визуальное напряжение: высокое
- акцент кадра: существенное ухудшение условий работы стропа
- ветви по цвету/толщине: заметно усилены
- характер остановки: более выраженное замедление перед стопом
- смысл: потеря `30%`, допустимая нагрузка `70%`

## Стоп-кадр `100°`

- высота груза: финальная, но с сохранением комфортного зазора до крюка
- видимая длина ветвей: минимальная в последовательности
- внутренний участок стропа: максимально уплотнен
- внутренняя намотка: максимальная
- внутреннее расстояние между ветвями: минимальное
- визуальное напряжение: максимальное
- акцент кадра: предельно неблагоприятный угол
- ветви по цвету/толщине: максимально усилены в пределах аккуратной учебной графики
- характер остановки: плавное затухание и чистая фиксация
- смысл: потеря `40%`, допустимая нагрузка `60%`, критично

## Пост-эпизод после `100°`

- задержка после стопа: `2-3 секунды`
- положение груза: еще горизонтальное, без перекоса
- положение крюка: неподвижно
- ветви стропа: сначала удерживают нагрузку, затем начинают мелко вибрировать
- характер вибрации: короткая, нарастающая, высокочастотная
- внутренний участок стропа: визуально самый напряженный
- момент разрыва: быстрый, резкий
- звук: хлопок
- визуальный смысл: критическая перегрузка переходит в разрушение стропа

## Как должна ощущаться динамика всей сцены

- `30° -> 45°`: изменение мягкое, почти обучающее
- `45° -> 60°`: рост напряжения становится заметным
- `60° -> 90°`: переломный участок, ухудшение должно чувствоваться сильнее
- `90° -> 100°`: финальное усиление, но без перегрузки кадра
- `100° -> разрыв`: короткая пауза, вибрация, резкий финальный срыв

## Что особенно важно не сломать при сборке

- не уводить груз с центральной оси;
- не допускать перекоса груза;
- не перемещать точки контакта на грузе;
- не делать финал слишком тесным;
- не превращать микроэффект натяжения в грубую деформацию стропа.
- не растягивать сцену разрыва слишком долго;
- не делать тяжелую аварийную детализацию;
- не терять учебную ясность в момент хлопка.
