# Master Package V2

## Назначение

Этот пакет фиксирует собранную версию ролика про влияние угла между ветвями стропа на допустимую нагрузку.

## Состав пакета

- `angle-between-branches_master-v2_2026-06-26.mp4` — текущий собранный мастер-видео.
- `angle-between-branches_poster-v2_2026-06-26.png` — постер/контрольный кадр.
- `angle-between-branches_voice-male-dmitry-v2_2026-06-26.mp3` — утвержденная мужская озвучка.
- `angle-between-branches_subtitles-v2_2026-06-26.vtt` — субтитры по таймингу.
- `angle-between-branches_timing-plan-v2_2026-06-26.txt` — технический план длительностей и этапов.
- `angle-between-branches_clean-frames-contact-sheet-v4_2026-06-26.png` — контрольная сетка стоп-кадров.

## Что зафиксировано в этой версии

- Последовательность углов: `30° -> 45° -> 60° -> 90° -> 100°`.
- Угол трактуется как угол между ветвями.
- Крюк неподвижен.
- Груз поднимается ровно, без перекоса.
- Центр груза идет по одной вертикали.
- Точки крепления на грузе остаются фиксированными.
- На `30°` точки крепления стоят на расстоянии `25%` от краев груза.
- Намотка идет от точек крепления к центру.
- После `100°` сохранены пауза, вибрация и разрыв с хлопком.
- Используется утвержденная мужская озвучка `dmitry`.

## Согласованные цифры

- `30°`: потеря `0%`, допустимая нагрузка `100%`
- `45°`: потеря `10%`, допустимая нагрузка `90%`
- `60°`: потеря `15%`, допустимая нагрузка `85%`
- `90°`: потеря `30%`, допустимая нагрузка `70%`
- `100°`: потеря `40%`, допустимая нагрузка `60%`, критично
