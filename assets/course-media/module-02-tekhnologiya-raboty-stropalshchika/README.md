# Media Pack: Module 02

Сюда складываем визуальные материалы для `Модуль 2. Технология работы стропальщика`.

## Папки

- `images` - фото рабочих ситуаций, складирования, сопровождения груза и сигнализации.
- `diagrams` - алгоритмы, чек-листы, схемы опасных зон и технологических циклов.
- `video` - ролики и короткие фрагменты, если появятся.
- `references` - ссылки, исходники, служебные описания медиа.

## Первый приоритет

- ручные сигналы стропальщика;
- подготовка рабочего места;
- сопровождение груза оттяжками;
- схемы безопасного складирования;
- ошибки при расстроповке и установке груза.
