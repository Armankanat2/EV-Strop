# База знаний

Папка для нормализованного знания по теме стропальщика.

Рекомендуемое назначение подпапок:

- `cards/` - карточки терминов, процессов, правил и ошибок;
- `images/` - изображения, схемы и их описание;
- `sources/` - список источников, выписки и ссылки на локальные материалы.

Если рядом уже есть внешняя база знаний по профессии, сюда стоит переносить только очищенные и согласованные материалы, а не весь сырой массив без разбора.
