## Как тебе пользоваться этой таблицей

### Шаг 1. Понять, где именно проблема

Примеры:

- не нравится новая раскладка папок;
- пропал документ;
- сломались переходы в PowerPoint;
- не та озвучка;
- нужно вернуть старую логику изменения слайдов;
- нужно посмотреть, что было раньше.

### Шаг 2. Найти подходящий коммит в таблице

Не нужно искать идеально точный коммит.

Нужно найти самый близкий по смыслу.

### Шаг 3. Выбрать, что ты хочешь сделать

Есть 3 нормальных сценария:

1. посмотреть, что было в коммите;
2. сравнить текущий проект с коммитом;
3. откатить файл, папку или часть проекта к коммиту.

### Шаг 4. Просто написать мне готовой фразой

## Готовые фразы для тебя

### Если хочешь просто посмотреть

```text
Покажи мне, что изменилось в коммите [хеш]. Объясни простыми словами, без кода.
```

Пример:

```text
Покажи мне, что изменилось в коммите d6ce266. Объясни простыми словами, без кода.
```

### Если хочешь сравнить с текущим состоянием

```text
Сравни текущий проект с коммитом [хеш] и коротко скажи, что изменилось.
```

Пример:

```text
Сравни текущий проект с коммитом eaf0848 и коротко скажи, что изменилось.
```

### Если хочешь откатить один файл

```text
Откати только файл [путь_к_файлу] к состоянию из коммита [хеш]. Сначала покажи, что изменится.
```

Пример:

```text
Откати только файл docs/README.md к состоянию из коммита eaf0848. Сначала покажи, что изменится.
```

### Если хочешь откатить одну папку

```text
Откати только папку [путь_к_папке] к состоянию из коммита [хеш]. Сначала покажи план и список файлов.
```

Пример:

```text
Откати только папку docs/00_навигация_и_изменения к состоянию из коммита 0e00147. Сначала покажи план и список файлов.
```

### Если хочешь откатить весь блок изменений

```text
Нужно вернуть блок изменений, связанный с коммитом [хеш]. Сначала покажи, что именно вернется назад.
```

Пример:

```text
Нужно вернуть блок изменений, связанный с коммитом dbc6c4d. Сначала покажи, что именно вернется назад.
```

### Если не знаешь, какой коммит выбрать

```text
Помоги выбрать коммит для отката. Проблема такая: [кратко опиши проблему].
```
