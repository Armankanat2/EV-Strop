# Боевая памятка: 4 короткие команды

Дата фиксации: `2026-06-20`

Копируй одну строку, меняй значения в квадратных скобках и отправляй мне.

1. Добавить линейный слайд:
`Добавь новый линейный слайд между [Sxxx] и [Sxxx]. Название: [название]. Причина: [причина]. Тест: [нет / какой]. Проведи изменение по системе проекта и обнови все связанные документы.`

2. Убрать линейный слайд:
`Убери линейный слайд [Sxxx]. Причина: [причина]. Содержание: [удалить / перенести в Sxxx]. Тест: [нет / какой]. Проведи изменение по системе проекта и обнови все связанные документы.`

3. Разделить один слайд на два:
`Раздели слайд [Sxxx] на два. Первый: [название 1]. Второй: [название 2]. Причина: [причина]. Тест: [нет / какой]. Проведи изменение по системе проекта и обнови все связанные документы.`

4. Добавить подвальный слайд:
`Добавь вынос от [Sxxx]. Тип: [P / PP / P56]. Содержание: [что должно быть на выносе]. Причина: [причина]. Если это PP: [из какого слайда открывается / не применимо]. Обнови связи, возврат и связанные документы проекта.`

## Самое короткое правило

- меняется основной маршрут — используй команды `1`, `2` или `3`;
- основной маршрут не меняется, нужен только вынос — используй команду `4`.
