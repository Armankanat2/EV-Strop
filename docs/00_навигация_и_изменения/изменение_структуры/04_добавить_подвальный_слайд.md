# Промт: добавить подвальный слайд

## Когда использовать

Используй этот промт, если основной линейный маршрут не меняется, но нужен вынесенный слайд:

- `P` — обычный подвал;
- `PP` — второй уровень вложенности;
- `P56` — вынос для `5-6 разряда`.

Пример:

- нужен отдельный термин;
- нужна схема к одному экрану;
- нужен дополнительный вынос для `5-6 разряда`;
- нужен вложенный слайд из уже существующего подвала.

## Что заполнить

- `родительский слайд`: `[например: S015]`
- `тип выноса`: `[P / PP / P56]`
- `что должно быть на слайде`: `[например: схема центра тяжести]`
- `причина`: `[почему нельзя оставить это в основном слайде]`
- `если это PP`: `[из какого именно слайда открывается]`

## Готовый промт

Скопируй текст ниже целиком и замени значения в квадратных скобках.

```text
Мы работаем в проекте `ПРОЕКТ ЭВ СТРОПАЛЬЩИК`.

Нужно добавить вынесенный слайд без изменения основного линейного маршрута.

Исходные данные:
- родительский слайд: [S015]
- тип выноса: [P / PP / P56]
- что должно быть на слайде: [содержание]
- причина: [почему нужен отдельный вынос]
- если это PP, из какого именно слайда открывается: [например: S015-P01 / не применимо]

Что нужно сделать:
1. Добавь новый вынесенный слайд по принятой схеме нумерации проекта.
2. Не меняй основной линейный маршрут, если в этом нет необходимости.
3. Обнови возвраты и дочерние связи.
4. Если нужно, обнови связанные таблицы, реестры и production-документы.
5. Внеси запись в журнал нумерации.
6. В конце кратко напиши:
   - какой новый `ID` создан;
   - какие файлы были обновлены;
   - что мне стоит проверить руками.

Если каких-то данных не хватает, сделай разумные рабочие допущения и явно укажи их в ответе.
```
