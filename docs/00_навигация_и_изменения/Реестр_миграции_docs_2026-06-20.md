# Реестр миграции `docs`

Дата фиксации: `2026-06-20`

## Назначение

Этот документ фиксирует реорганизацию папки `docs/` в проекте `ЭВ Стропальщик`.

Он нужен, чтобы:

- быстро понимать, куда переехал старый документ;
- сверять старые ссылки и новые пути;
- не терять контекст после перестройки структуры папок;
- упрощать проверку `git diff` и последующие переносы.

## Базовое правило миграции

Корень `docs/` больше не используется как плоский список файлов.

Теперь документы разложены по разделам:

- `docs/00_навигация_и_изменения/`
- `docs/01_структура_курса/`
- `docs/02_контент_модулей/`
- `docs/03_тесты_и_разбор_ошибок/`
- `docs/04_production_и_реестры/`
- `docs/05_концепты_и_согласование/`
- `docs/06_нпа_и_источники/`
