## `S002` Какие документы регулируют работу стропальщика

**Что видит пользователь на экране**

- Работу стропальщика регулируют:
- Трудовое законодательство
- Охрана труда
- Промышленная безопасность
- Инструкции работодателя

Нижняя подпись:

- Стропальщик работает по установленным правилам, а не `по памяти` и не `по привычке`.

**Как выглядит**

- схема из `4` равных блоков;
- иконки `закон / каска / кран / инструкция`;
- внизу короткая строка-вывод.

**Переходы**

- `S002` -> `S003`
- `Трудовое законодательство` -> `S002-P01`
- `Охрана труда` -> `S002-P02`
- `Промышленная безопасность` -> `S002-P03`
- `Инструкции работодателя` -> `S002-P04`

## `S003` Кто такой стропальщик

**Что видит пользователь на экране**

- Стропальщик:
- Принимает и оценивает груз
- Выбирает строп и способ строповки
- Подает сигналы при перемещении
- Следит за безопасностью работы

**Как выглядит**

- фото рабочего процесса;
- простая схема:
  `груз -> строповка -> сигнал -> перемещение -> установка`

**Переходы**

- `S003` -> `S004`
- `Что делает стропальщик по этапам` -> `S003-P01`

## `S004` Кто может работать стропальщиком

**Что видит пользователь на экране**

- Может работать стропальщиком тот, кто:
- Прошел обучение
- Прошел инструктаж
- Прошел проверку знаний
- Допущен к работе
- Использует СИЗ

**Как выглядит**

- чек-лист допуска;
- иконки `обучение / инструктаж / проверка / допуск / СИЗ`.

**Переходы**

- `S004` -> `S005`
- `Что такое инструктаж` -> `S004-P01`
- `Что такое проверка знаний` -> `S004-P02`
- `Что входит в СИЗ` -> `S004-P03`

---

## `S005` Разряды `2-4`

**Что видит пользователь на экране**

- Разряды, которые закрывает курс:
- `2 разряд` — простые грузы до `5 т`
- `3 разряд` — простые грузы свыше `5 до 25 т`
- `4 разряд` — сложные грузы до `5 т`

Нижняя подпись:

- Этот курс дает базу для работы стропальщика `2-4 разряда`.

**Как выглядит**

- короткая таблица из `3` строк;
- цветовое деление по разрядам;
- крупная плашка `Стропальщик 5-6 разряд`.

**Переходы**

- `S005` -> `S006`
- `Как понимать простые и сложные грузы` -> `S005-P01`
- `Стропальщик 5-6 разряд` -> `S005-P56-01`
