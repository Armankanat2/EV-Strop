# Реестр слайдов разбора ошибок по промежуточным тестам курса `ЭВ Стропальщик`

Дата фиксации: `2026-06-18`

## Назначение

Это единый рабочий реестр для всех слайдов разбора ошибки в промежуточных тестах курса.

Он нужен, чтобы держать в одной таблице:

- связь `тест -> вопрос -> слайд разбора`;
- логику возврата обратно в тест;
- краткое содержание поясняющего слайда;
- резерв под будущие тестовые блоки.

## Базовое правило

- разбор ошибки используется только в промежуточных тестах `S018`, `S031`, `S044`, `S051`;
- итоговые тестовые слайды `S053`, `S054`, `S055` в этот реестр не входят;
- для разбора ошибки используем обычный `подвал` родительского теста;
- обратная ссылка всегда возвращает в исходный тестовый слайд.

## Рабочая схема нумерации

Базовая привязка на текущем этапе принимается простой и прозрачной:

- `S018-Q01 -> S018-P01`
- `S018-Q02 -> S018-P02`
- `S031-Q01 -> S031-P01`
- `S044-Q01 -> S044-P01`

Если позже один слайд разбора будет обслуживать несколько однотипных ошибок, это можно оставить в реестре отдельной пометкой, но стартовая схема сейчас идет по принципу `один вопрос = один слайд разбора`.

## Практический вывод

Теперь у курса есть отдельный рабочий реестр, где можно по каждой ошибке видеть:

- из какого теста и вопроса идет переход;
- какой `ID` получает подвальный слайд разбора;
- что именно этот слайд должен объяснить;
- куда именно он возвращает пользователя.

Следующий удобный шаг после этого документа — собрать для этих слайдов визуальный реестр `схема / картинка / знак / видео-фрагмент`.
