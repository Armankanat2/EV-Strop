# Общий мастер-реестр разбора ошибок: промежуточные тесты курса

Дата фиксации: `2026-06-18`

## Назначение

Это единый мастер-реестр по всем промежуточным тестам курса `ЭВ Стропальщик`:

- `S018` — Тема `2.1. Строповка грузов`;
- `S031` — Тема `2.2. Технология работы стропальщика`;
- `S044` — Тема `2.3. Производство работ`;
- `S051` — Тема `3. Охрана труда`.

Документ собирает в одном месте:

- `тест`;
- `вопрос`;
- `слайд разбора ошибки`;
- `экранный тезис`;
- `смысл озвучки`;
- `визуальное решение`;
- `рабочие файлы`;
- `возврат в тест`.

## Как использовать реестр

- методист проверяет, что каждый неверный ответ получает точное пояснение;
- дизайнер видит, какой тип визуала нужен на конкретный слайд;
- продакшн видит основные и дополнительные файлы;
- сборщик курса видит обратную ссылку в родительский тест;
- руководитель проекта видит весь объем промежуточных разборов в одном документе.

## Сводка по пакету

- всего промежуточных тестов с разбором ошибок: `4`;
- всего вопросов с отдельными слайдами разбора: `34`;
- из них:
  - `12` в `S018`;
  - `12` в `S031`;
  - `5` в `S044`;
  - `5` в `S051`.
