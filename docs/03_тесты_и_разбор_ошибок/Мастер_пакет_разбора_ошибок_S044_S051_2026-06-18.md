# Мастер-пакет разбора ошибок `S044` и `S051`

Дата фиксации: `2026-06-18`

## Назначение

Это единый рабочий документ по подвальным слайдам разбора ошибок для промежуточных тестов:

- `S044` — Тема `2.3. Производство работ`;
- `S051` — Тема `3. Охрана труда`.

Документ собирает в одной связке:

- `вопрос`;
- `слайд разбора`;
- `экранный текст`;
- `акцент озвучки`;
- `визуальное решение`;
- `рабочие файлы`;
- `возврат в тест`.

## Как использовать документ

- методист видит, что именно должен объяснить слайд;
- дизайнер видит тип визуала и главный смысл кадра;
- продакшн видит, какой файл является основным, а какой резервным;
- сборщик курса видит, куда должен возвращать слайд после разбора ошибки.

## Что уже готово по пакету

- логика `вопрос -> ошибка -> пояснение -> возврат` зафиксирована;
- тексты экранов и акценты озвучки подготовлены;
- визуальные решения по всем `10` слайдам выбраны;
- основные production-файлы названы и привязаны к конкретным ID.

## Что остается продакшну

- изготовить основные `.png` по каждому слайду;
- подготовить редактируемые `.pptx` там, где они уже предусмотрены;
- решить по ходу сборки, где реально нужен `PP01`, а где можно обойтись без него;
- после изготовления обновить статусы в production-реестре файлов.

## Практический вывод

Теперь по `S044` и `S051` есть единый мастер-пакет, где можно пройти всю цепочку без переходов между отдельными документами:

`вопрос -> слайд -> текст -> визуал -> файл -> возврат`.
