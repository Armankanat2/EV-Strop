# Тексты слайдов разбора ошибок `S045-P01...P05` и `S052-P01...P05`

Дата фиксации: `2026-06-18`

## Назначение

Это рабочий документ с готовыми текстами для подвальных слайдов разбора ошибок в промежуточных тестах:

- `S045` — Тема `2.3. Производство работ`;
- `S052` — Тема `3. Охрана труда`.

Формат такой же, как в предыдущих пакетах:

- `Экран` — короткий текст для показа на слайде;
- `Акцент озвучки` — пояснение для диктора;
- `Кнопка` — обязательная обратная ссылка в тест.

## Тест `S045` — Тема `2.3. Производство работ`

### `S045-P01` Ошибка: опасная зона не определена до начала работы

Экран:

- Опасную зону определяют заранее
- До начала перемещения груза
- Людей в этой зоне быть не должно
- Без этого работу начинать нельзя

Акцент озвучки:

- Опасная зона существует всегда, и стропальщик обязан понимать ее до начала перемещения груза, а не после возникновения риска.

Кнопка:

- `Вернуться к тесту`

### `S045-P02` Ошибка: опасная обстановка оценивается слишком узко

Экран:

- Смотри не только на груз
- Оценивай траекторию
- Учитывай раскачивание и помехи
- Думай о людях вокруг

Акцент озвучки:

- Опасная обстановка складывается из нескольких факторов сразу: движение груза, пространство, препятствия и присутствие людей.

Кнопка:

- `Вернуться к тесту`

### `S045-P03` Ошибка: работа продолжается без безопасных условий

Экран:

- Нет безопасных условий
- Работу нужно прекратить
- Нельзя работать без понятной организации
- Риск сначала устраняют

Акцент озвучки:

- Если безопасные условия не обеспечены, стропальщик обязан остановить работу, даже если операция уже подготовлена.

Кнопка:

- `Вернуться к тесту`

Экран:

- ЛЭП — особая опасность
- Нельзя оценивать расстояние "на глаз"
- Нужны специальные меры
- Нужен организованный порядок работ

Акцент озвучки:

- Работа вблизи ЛЭП допускается только при понятных дистанциях, специальных мерах безопасности и четкой организации работ.

Кнопка:

- `Вернуться к тесту`

### `S045-P05` Ошибка: работа у ЛЭП продолжается при неясных дистанциях

Экран:

- Дистанции неясны
- Схема работ не разъяснена
- Работу нужно остановить
- Безопасные условия сначала уточняют

Акцент озвучки:

- При работе рядом с ЛЭП недопустимо продолжать операцию, если участникам не ясны границы безопасной зоны и порядок действий.

- `Вернуться к тесту`

## Тест `S052` — Тема `3. Охрана труда`

### `S052-P01` Ошибка: охрана труда воспринимается слишком узко

Экран:

- Охрана труда — это не формальность
- Это безопасное рабочее место
- Это СИЗ и правила
- Это право остановить опасную работу

Акцент озвучки:

- Для стропальщика охрана труда означает ежедневное безопасное поведение, а не только прохождение инструктажа.

Кнопка:

- `Вернуться к тесту`

### `S052-P02` Ошибка: обязанности на рабочем месте понимаются неполно

Экран:

- Используй СИЗ
- Выполняй инструкции
- Сообщай об опасности
- Не продолжай работу при риске

Акцент озвучки:

- Работник обязан не просто присутствовать на рабочем месте, а действовать по правилам и не скрывать опасные условия.

Кнопка:

- `Вернуться к тесту`

### `S052-P03` Ошибка: допуск к работе понимается неправильно

Экран:

- Допуск не дается автоматически
- Сначала обучение
- Потом инструктаж и проверка знаний
- Только после этого работа

Акцент озвучки:

- Законный допуск к работе строится по последовательной цепочке, а не по стажу, спешке или устной договоренности.

Кнопка:

- `Вернуться к тесту`

### `S052-P04` Ошибка: при опасности действия откладываются

Экран:

- Опасность требует немедленного действия
- Останови работу
- Предупреди людей
- Сообщи ответственному лицу

Акцент озвучки:

- При опасной или аварийной ситуации нельзя сначала "доделать" операцию. Сначала останавливают опасное действие и предупреждают окружающих.

Кнопка:

- `Вернуться к тесту`

### `S052-P05` Ошибка: неисправность и люди в зоне не приводят к остановке

Экран:

- Есть неисправность оснастки
- Люди находятся в зоне работ
- Работу нужно остановить сразу
- Опасность нельзя игнорировать

Акцент озвучки:

- Если одновременно есть техническая неисправность и люди в опасной зоне, это прямое основание для немедленной остановки работ.

Кнопка:

- `Вернуться к тесту`

## Практический вывод

Теперь для `S045` и `S052` есть готовая текстовая база:

- короткий экранный текст;
- акцент для озвучки;
- единая кнопка возврата в тест.

Это закрывает все промежуточные тесты курса по контуру `вопрос -> ошибка -> пояснение -> возврат`.
