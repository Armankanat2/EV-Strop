# Реестр тестовых вопросов курса `ЭВ Стропальщик`

Дата фиксации: `2026-06-18`

## Назначение

Это единый рабочий реестр тестовых вопросов по курсу.

Он ведется в формате:

- `ID теста`
- `ID вопроса`
- `Текст вопроса / кейса`
- `Правильный ответ`
- `Неверные варианты`
- `Нужен ли слайд разбора ошибки`

## Правило по разбору ошибки

- для промежуточных тестов `S019`, `S032`, `S045`, `S052` поле `Нужен ли слайд разбора ошибки` может быть `да`;
- для итогового тестового блока `S054`, `S055`, `S056` это поле фиксируем как `нет`, если вопрос относится именно к итоговому контролю;
- для `S056` переход на поясняющий слайд при ошибке не используется.
