# Реестр тестовых вопросов курса `ЭВ Стропальщик`

Дата фиксации: `2026-06-18`

## Назначение

Это единый рабочий реестр тестовых вопросов по курсу.

Он ведется в формате:

- `ID теста`
- `ID вопроса`
- `Текст вопроса / кейса`
- `Правильный ответ`
- `Неверные варианты`
- `Нужен ли слайд разбора ошибки`

## Правило по разбору ошибки

- для промежуточных тестов `S027`, `S040`, `S053`, `S060` поле `Нужен ли слайд разбора ошибки` может быть `да`;
- для итогового тестового блока `S062`, `S063`, `S064` это поле фиксируем как `нет`, если вопрос относится именно к итоговому контролю;
- для `S064` переход на поясняющий слайд при ошибке не используется.
