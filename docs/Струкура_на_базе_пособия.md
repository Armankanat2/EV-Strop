## Логика структуры

За основу взято оглавление пособия. Структура курса сохраняет исходную последовательность тем, чтобы:

- не ломать логику базового материала;
- упростить перенос текстовки из пособия в онлайн-формат;
- сразу разложить курс на модули, уроки и контрольные точки.

## Предлагаемая структура курса

## Модуль 1. Строповка грузов

Назначение модуля:
дать базовое понимание грузов, их особенностей и правил правильной строповки.

Уроки модуля:

1. Введение в работу стропальщика
2. Сведения о грузах
3. Масса, габариты, маркировка и устойчивость грузов
4. Правила строповки грузов
5. Типовые ошибки при строповке

Основа из пособия:

- Глава 1. Строповка грузов
- 1.1. Сведения о грузах
- 1.2. Правила строповки

Результат изучения:

- слушатель понимает, какие бывают грузы;
- различает основные характеристики груза, влияющие на подъем и перемещение;
- знает общие правила строповки;
- понимает, к чему приводят ошибки при выборе способа строповки.

## Модуль 2. Технология стропальных работ

Назначение модуля:
показать порядок работы стропальщика до начала работ, во время перемещения груза и после завершения операций.

Уроки модуля:

1. Обязанности стропальщика
2. Подготовка к работе и проверка условий производства работ
3. Перемещение грузов кранами
4. Сигналы между стропальщиком и машинистом крана
5. Складирование грузов

Основа из пособия:

- Глава 2. Технология стропальных работ
- 2.1. Обязанности стропальщика
- 2.2. Перемещение грузов кранами
- 2.3. Складирование грузов

Результат изучения:

- слушатель знает обязанности стропальщика до, во время и после работы;
- понимает порядок безопасного перемещения грузов;
- знает требования к складированию;
- ориентируется в типовых сигналах и взаимодействии с крановщиком.

## Модуль 3. Производство работ и опасные условия

Назначение модуля:
разобрать организацию работ, опасные зоны и специальные условия, при которых риски особенно высоки.

Уроки модуля:

1. Опасные зоны при производстве стропальных работ
2. Требования безопасности при производстве работ
3. Запрещенные действия при подъеме и перемещении грузов
4. Работа по ППР и технологическим картам
5. Производство работ стреловыми кранами вблизи ЛЭП

Основа из пособия:

- Глава 3. Производство работ
- 3.1. Опасные зоны
- 3.2. Требования безопасности при производстве работ
- 3.3. Производство работ стреловыми кранами вблизи ЛЭП

Результат изучения:

- слушатель умеет распознавать опасные зоны;
- знает базовые запреты и ограничения;
- понимает, когда работы выполняются по наряду-допуску;
- понимает повышенные риски при работе рядом с ЛЭП.

## Модуль 4. Охрана труда стропальщика

Назначение модуля:
собрать требования по охране труда в отдельный итоговый блок, который закрепляет безопасную модель поведения на рабочем месте.

Уроки модуля:

1. Охрана труда: общие сведения
2. Инструктажи и допуск к работе
3. Требования охраны труда на рабочем месте стропальщика
4. Действия при аварийных и нештатных ситуациях
5. Итоговое повторение по безопасности

Основа из пособия:

- Глава 4. Охрана труда
- 4.1. Охрана труда. Общие сведения
- 4.2. Охрана труда на рабочем месте стропальщика

Результат изучения:

- слушатель знает виды инструктажей;
- понимает требования допуска к работе;
- знает базовые действия при опасных и аварийных ситуациях;
- закрепляет требования безопасного поведения на рабочем месте.

## Итоговая схема курса

Утвержденная укрупненная схема:

1. Тема 1. Общая часть
2. Тема 2. Стропальные работы
3. Тема 3. Охрана труда
4. Тема 4. Итоговое тестирование

Внутренняя разбивка `Темы 2. Стропальные работы`:

1. Тема 2.1. Строповка грузов
2. Тема 2.2. Технология работ
3. Тема 2.3. Производство работ

Отдельная следующая тема:

1. Тема 3. Охрана труда

Контрольная логика:

1. Промежуточный тест после `Темы 2.1. Строповка грузов`
2. Промежуточный тест после `Темы 2.2. Технологии работ`
3. Промежуточный тест после `Темы 2.3. Производства работ`
4. Промежуточный тест после `Темы 3. Охрана труда`
5. Тема 4. Итоговое тестирование и квалификационный экзамен в финале

## Что важно учесть при дальнейшей разработке

- основную текстовку переносим из базового пособия;
- спорные или кратко раскрытые темы усиливаем материалами из учебника `Стропальные и такелажные работы`;
- обязательные требования, запреты и формулировки обязанностей сверяем с типовой инструкцией стропальщика;
- при последующей сборке контента каждый модуль желательно делить на короткие онлайн-уроки по 5-10 минут;
- итоговую проверку знаний лучше собирать по завершении каждого модуля и отдельным финальным тестом в конце курса.

## Рабочий вывод

Базовая структура курса может быть построена прямо по 4 главам пособия. Для онлайн-формата их удобно превратить в 4 модуля с последовательным переходом:

`грузы и строповка -> технология работы -> опасные условия -> охрана труда`

Эту структуру нужно использовать как опорную для следующих шагов: детальной поурочной разбивки, наполнения каждого раздела текстом, подготовки визуалов и сборки промежуточных тестов.
