# Чек-лист для съемочной группы. Концепт 2

Дата фиксации: `2026-06-15`

## Цель

Снять короткие, понятные и безопасные материалы для курса стропальщика по логике Типовой инструкции.

## Перед съемкой

- сверить номера слайдов и этап работы;
- назначить ответственного за безопасность;
- подготовить исправные грузы, стропы, захваты, тару, СИЗ;
- дефекты снимать только на выведенных из эксплуатации образцах;
- убрать лишних людей и лишние предметы из кадра;
- подготовить читаемые бирки, маркировку и паспорт стропа.

## Что снять обязательно

- проверка стропов, тары и захватов;
- бирка, маркировка, паспорт;
- подбор стропа под груз;
- выбраковка;
- где можно и где нельзя стропить;
- центр тяжести;
- пробный подъем;
- сигналы;
- сопровождение груза;
- оттяжки и багры;
- складирование и тара;
- работа у ЛЭП.

## На каждый важный блок

- общий план;
- средний план;
- крупный план действия, дефекта или сигнала;
- дубль `как правильно`;
- дубль `что не так`, если это безопасно.

## В кадре должно быть видно

- руки;
- точку зацепки;
- ветви стропа и угол;
- положение груза;
- безопасную позицию людей;
- дефект крупно;
- сигнал рукой полностью.

## Видео

- один ролик = одна мысль;
- длина `20-60 секунд`;
- старт сразу с ситуации;
- конец на правильном результате или `Стоп`;
- ошибки показывать без реального риска.

## Нельзя

- снимать опасное нарушение в динамике;
- поднимать груз ради красивого кадра;
- оставлять людей под грузом;
- использовать строп без бирки как правильный пример.

- по каждому приоритетному слайду есть ролик;
- по каждому сложному блоку есть крупный план;
- по дефектам есть библиотека фото;
- по сигналам есть библиотека фото и видео.
