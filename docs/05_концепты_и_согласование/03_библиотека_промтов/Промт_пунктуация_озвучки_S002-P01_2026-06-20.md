# Промт: пунктуация для озвучки `S002-P01`

Дата фиксации: `2026-06-20`

## Назначение

Это внутренний рабочий промт для подготовки дикторской версии текста по `S002-P01`.

Его задача:

- не переписывать смысл;
- не усложнять текст;
- расставлять понятные знаки препинания;
- делить длинные фразы на короткие;
- делать текст удобным для спокойной, уверенной озвучки.

## Готовый промт

```text
Возьми утвержденный текст озвучки для слайда `S002-P01`.

Твоя задача:
1. Ничего не менять по смыслу.
2. Не добавлять новую информацию от себя.
3. Упростить только ритм чтения.
4. Расставить знаки препинания так, чтобы текст было легко читать вслух.
5. Делить длинные фразы на короткие и понятные предложения.
6. Сохранять дружелюбный, уверенный, инструкторский тон.
7. Не использовать сложные слова, канцелярит и тяжелые обороты.
8. Делать текст удобным для взрослой рабочей аудитории.
9. Если одно предложение перегружено, разбивать его на два или три коротких.
10. Если нужна естественная пауза, использовать запятую, точку или двоеточие, но не перегружать текст знаками.

Формат ответа:
- сначала дай готовый текст для озвучки;
- каждая законченная мысль с новой строки;
- без комментариев;
- без объяснений;
- без списка правок.

Дополнительное правило:
- текст должен звучать так, как будто его читает спокойный инструктор;
- слушателю должно быть легко понять каждое действие с первого раза.
```

## Базовый текст, который сейчас принят для `S002-P01`

```text
Этот курс интерактивный.
Это значит, что вы не просто смотрите материал подряд.
Вы будете переходить по слайдам, открывать дополнительные пояснения и отвечать на вопросы.
Курс нужно проходить по порядку.
Сначала идет Тема 1. Потом Тема 2. Потом Тема 3. После этого идет итоговое тестирование.
Некоторые темы разбиты на более подробные блоки. Их тоже нужно проходить по порядку.
После тем и отдельных блоков будут тесты для самопроверки.
Если в слайде есть активная кнопка, нажмите на нее.
Так вы откроете дополнительную информацию: термин, картинку, схему или пояснение.
После просмотра нажмите `НАЗАД К КУРСУ` и продолжайте обучение.
Если в промежуточном тесте ответ неверный, откроется ссылка на пояснение.
Откройте ее, посмотрите материал, вернитесь к вопросу и продолжайте.
В конце курса будет итоговый тест.
В нем `20` вопросов.
Допускается не больше `3` ошибок.
В итоговом тесте подсказок и пояснений уже не будет.
Если тест не сдан, его можно пройти повторно.
При повторной попытке вопросы будут уже другими.
Этот курс дает базу для стропальщика `2-4 разряда`.
Если вы проходите подготовку на `5-6 разряд`, на отдельных слайдах увидите активную плашку `5-6 разряд - пройдите обучение`.
Если такая плашка есть, откройте ее и изучите дополнительный материал.
Для слушателя `2-4 разряда` этот дополнительный блок не обязателен.
```

## Как я буду это использовать

1. Беру утвержденный текст озвучки.
2. Прогоняю его через этот промт.
3. Получаю чистую дикторскую версию с удобной пунктуацией.
4. Только после этого отдаю текст в озвучку или в финальную сборку.
