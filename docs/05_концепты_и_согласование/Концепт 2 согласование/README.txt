Концепт 2. Папка согласования
Дата обновления: 2026-06-15

Назначение папки

В этой папке собраны последние docx-версии документов по Концепту 2 для согласования, обсуждения и передачи коллегам.

Что открывать в первую очередь

Если нужно быстро понять текущее состояние Концепта 2, лучше смотреть документы в таком порядке:

1. Концепт_2_структура_для_обсуждения_Типовая_инструкция_Игумнов_2026-06-15.docx
2. Концепт_2_приложение_по_подготовке_фото_и_видео_материалов_2026-06-15.docx
3. чек-листы по ролям

Описание файлов

Концепт_2_структура_презентации_по_типовой_инструкции_Игумнов_2026-06-15.docx

Базовая развернутая структура Концепта 2.

Что внутри:
- логика курса строго по последовательности Типовой инструкции;
- полный каркас презентации по этапам работы стропальщика;
- практическое усиление разделов материалом Игумнова;
- общая схема курса от допуска до ответственности.

Этот файл лучше использовать как опорную подробную версию структуры.

Концепт_2_структура_для_обсуждения_Типовая_инструкция_Игумнов_2026-06-15.docx

Основной рабочий документ для согласования.

Что внутри:
- разбивка курса по слайдам;
- сокращенная и прикладная версия Концепта 2;
- акцент на действиях, проверках, запретах и команде Стоп;
- список блоков, где особенно желательно видео.

Этот файл лучше использовать как главный документ для обсуждения состава курса.

Концепт_2_приложение_по_подготовке_фото_и_видео_материалов_2026-06-15.docx

Подробное приложение к основной структуре.

Что внутри:
- требования к подготовке фото и видео материалов;
- что именно снимать по ключевым слайдам;
- какие планы нужны;
- на чем делать акцент в кадре;
- требования по безопасности постановки;
- рекомендации по библиотеке дефектов, сигналов и файловой структуре.

Этот файл нужен для подготовки съемки и постановки задач съемочной группе.

Чек-лист_для_съемочной_группы_Концепт_2_2026-06-15.docx

Короткий рабочий чек-лист для всей съемочной команды.

Что внутри:
- что проверить перед съемкой;
- что обязательно снять;
- что должно быть видно в кадре;
- базовые запреты и финальная проверка.

Этот файл нужен для общего использования на площадке.

Чек-лист_для_методиста_Концепт_2_2026-06-15.docx

Короткий чек-лист для методиста.

Что внутри:
- проверка учебной логики;
- контроль понятности материала;
- контроль последовательности этапов Типовой инструкции;
- проверка пригодности материалов для PowerPoint и iSpring.

Этот файл нужен для методической приемки материалов.

Чек-лист_для_оператора_Концепт_2_2026-06-15.docx

Короткий чек-лист для оператора.

Что внутри:
- что обязательно снять по планам;
- какие крупные планы нельзя пропустить;
- на каких блоках нужен особый акцент;
- базовые требования к видео и именованию файлов.

Этот файл нужен для работы оператора на съемке.

Чек-лист_для_преподавателя_озвучки_Концепт_2_2026-06-15.docx

Короткий чек-лист для преподавателя, который будет делать озвучку.

Что внутри:
- как говорить коротко и понятно;
- на чем делать акцент в речи;
- какие формулировки лучше избегать;
- как проверять, что озвучка помогает визуалу, а не дублирует его.

Этот файл нужен при записи закадрового текста и при подготовке дикторских формулировок.
