# Чек-лист для съемочной группы. Концепт 1

Дата фиксации: `2026-06-15`

## Перед съемкой

- сверить номера слайдов и приоритетные блоки;
- назначить ответственного за безопасность;
- подготовить исправные грузы, стропы, захваты, СИЗ;
- дефекты снимать только на выведенных из эксплуатации образцах;
- убрать из кадра лишних людей и лишние предметы;
- подготовить читаемые бирки, маркировку, паспорт стропа.

## Снять обязательно

- оценка груза;
- правильные точки строповки;
- ошибки без реального риска;
- центр тяжести и пробный подъем;
- длинномерный груз;
- виды стропов и захватов;
- бирка, маркировка, паспорт;
- дефекты и выбраковка;
- крюк, замок, коуш, звено, траверса;
- сигналы стропальщика, сигнальщик, старший стропальщик;
- полный цикл одной операции;
- опасная зона и риск у ЛЭП.

## На каждый важный блок

- общий план;
- средний план;
- крупный план действия, дефекта или жеста;
- дубль `как правильно`;
- дубль `что не так`, если это безопасно.

## В кадре должно быть видно

- руки;
- точка зацепки;
- положение ветвей и угол;
- положение груза;
- безопасная позиция людей;
- дефект крупно;
- сигнал рукой полностью.

## Видео

- один ролик = одна мысль;
- длина `20-60 секунд`;
- без рывков камеры;
- старт сразу с ситуации;
- конец с правильным результатом или `Стоп`.

## Нельзя

- снимать реальное опасное нарушение в динамике;
- поднимать груз ради красивого кадра;
- оставлять людей под грузом;
- использовать строп без бирки как правильный пример.

## Финальная проверка

- по каждому приоритетному слайду есть ролик;
- по каждому сложному блоку есть крупный план;
- по дефектам есть библиотека фото;
- по сигналам есть библиотека фото и видео.
