# Рабочая карта наполнения: Тема 1 `Общая часть`

Дата фиксации: `2026-06-20`

## Назначение

Это прикладной production-документ по первому блоку курса.

Он нужен, чтобы по `Теме 1` сразу было видно:

- что идет в линейной последовательности;
- что уходит в `подвал`;
- что уходит в `подподвал`;
- какой материал нужен для визуализации;
- что остается на экране;
- что уходит в озвучку;
- как устроена навигация.

## Линейный маршрут блока

Основная линейка блока:

`S001 -> S002 -> S003 -> S004 -> S005 -> S006 -> S007`

Это обязательный путь для каждого слушателя.

## Что считаем обязательным в первой сборке

Для первого рабочего прохода обязательны:

- `S001`
- `S002`
- `S003`
- `S004`
- `S005`
- `S006`

Желательно сразу подготовить:

- `S004-P01`
- `S005-P03`
- `S006-P56-01`

Остальные выносы можно подключать вторым проходом, если нужно упростить первую сборку.

Отдельное правило:

- `S002-P01` не считаем ранним обязательным медиа-элементом;
- для текущей рабочей сборки по `S002-P01` берем мужскую озвучку `S002-P01_ozvuchka_55_sec_stress_2026-06-20.mp3`;
- его финальный ролик собираем последним внутри вводного блока.

## Короткий вывод по блоку

После изменения структуры первый блок больше не начинается сразу с нормативной темы.

Теперь логика такая:

- `S001` — вход в курс;
- `S002` — быстрое объяснение, как пользоваться курсом и где смотреть термины;
- `S003-S006` — содержательная часть вводного блока;
- `S007` — переход в тему 2.
