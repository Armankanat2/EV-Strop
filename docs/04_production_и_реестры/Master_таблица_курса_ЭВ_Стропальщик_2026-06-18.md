## Как использовать master-таблицу

1. Сначала собираем линейный маршрут `S001-S065`.
2. Потом добавляем только те выносы `подвала`, которые реально нужны для пояснения.
3. Контур `5-6 разряд` подключаем точечно через уже отмеченные плашки.
4. Если нужен следующий производственный шаг, на основе этой таблицы можно делать отдельный реестр `озвучка / визуал / видео / тест`.
