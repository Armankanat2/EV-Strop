## Базовые правила сборки

1. Основной обязательный маршрут курса идет строго по линии `S001-S064`.
2. `P`-слайды используются как поясняющие подвалы и всегда возвращают в родительский слайд.
3. `P56`-слайды относятся только к дополнительному контуру `5-6 разряд` и не заменяют базовый маршрут `2-4 разряда`.
4. `PP`-слайды открываются только из `P56` или из перегруженного справочного подвала и считаются вторым уровнем вложенности.
5. Для тестов `S026`, `S039`, `S052`, `S059` используем подвалы разбора ошибок с возвратом в тест.
6. Для `S063` подвалы разбора ошибки не используем.

## Ограничение на текущий проход

На текущем этапе действует жесткое правило по линейной структуре:

- участок `S001-S012` считаем замороженным;
- до слайда `S012` включительно линейную структуру не меняем;
- любые новые изменения линейного маршрута, добавление, перенос, разделение или удаление линейных экранов разрешены только начиная с `S013`.
