# Production-реестр файлов: визуалы разбора ошибок `S018` и `S031`

Дата фиксации: `2026-06-18`

## Назначение

Это рабочий production-реестр файлов для визуалов подвальных слайдов разбора ошибок:

- `S018-P01...P12`;
- `S031-P01...P12`.

Документ нужен, чтобы на уровне файлов сразу закрепить:

- какой файл нужен на каждый слайд;
- какой тип файла лучше использовать;
- что уже идет в первую очередь;
- откуда логичнее брать исходный материал;
- где нужен вложенный `PP`-файл второго уровня.

## Правило именования

Имена файлов строим от `ID` слайда по уже принятому шаблону:

- основной визуал: `S018-P03_tsentr-tyazhesti_diagram.png`
- вложенный визуал: `S018-P03-PP01_tsentr-tyazhesti_detail.png`

Рабочее правило:

- основной статичный визуал: чаще `png`;
- редактируемая схема: можно держать мастер в `pptx` или `svg`;
- если нужен отдельный вложенный материал, он получает `PP` в имени файла.
