# Каркас iSpring-курса `ЭВ Стропальщик`

Дата фиксации: `2026-06-16`

## Назначение

Этот документ переводит уже собранную предметную базу проекта в рабочий каркас для сборки курса в `iSpring`.

Задача каркаса:

- зафиксировать состав курса;
- определить, из каких модулей и уроков он состоит;
- показать, что именно нужно готовить в первую очередь;
- сделать структуру удобной для сборки в PowerPoint + iSpring.

## Формат курса

Рекомендуемый формат:

- самостоятельный электронный курс;
- прохождение по модулям;
- короткие экранные блоки вместо длинных текстов;
- мини-проверки после уроков;
- итоговый тест после завершения курса.

## Состав курса

1. Вводный блок
2. Модуль 1. `Строповка грузов`
3. Модуль 2. `Технология работы стропальщика`
4. Модуль 3. `Работы в опасных и специальных условиях`
5. Модуль 4. `Охрана труда и личная ответственность`
6. Итоговый контроль
7. Глоссарий и полезные материалы

## Рекомендуемая логика внутри каждого модуля

Для каждого урока используем одну и ту же модель:

1. Вход в тему
2. Что нужно знать
3. Основные правила
4. Алгоритм действий
5. Ошибки и запреты
6. Практический пример
7. Короткий вывод
8. Мини-тест

## Матрица сборки курса

### Вводный блок

Цель:
быстро объяснить роль стропальщика, риски работы и логику курса.

Интерактив:

- титульный экран;
- карта курса;
- входной вопрос или самопроверка.

### Модуль 1. `Строповка грузов`

Основа:
`Игумнов`, глава 1.

Уроки:

1. Сведения о грузах
2. Стропы и грузозахватные приспособления
3. Правила строповки
4. Проверка схемы строповки
5. Типовые схемы и ошибки

Интерактив:

- карточки;
- сопоставление;
- `правильно / неправильно`;
- мини-кейсы по ошибкам.

### Модуль 2. `Технология работы стропальщика`

Основа:
`Игумнов`, глава 2.

Уроки:

1. Подготовка к работе
2. Обязанности стропальщика
3. Сигнализация и взаимодействие
4. Перемещение и сопровождение груза
5. Складирование и расстроповка

Интерактив:

- пошаговые алгоритмы;
- вкладки;
- сценарии выбора действий.

### Модуль 3. `Работы в опасных и специальных условиях`

Основа:
`Игумнов`, глава 3.

Уроки:

1. Опасные зоны
2. Основные требования безопасности
3. Специальные условия работ
4. Работа вблизи ЛЭП
5. Ситуационные кейсы

Интерактив:

- разбор нарушений;
- сценарии остановки работ;
- кейсы с выбором решения.

### Модуль 4. `Охрана труда и личная ответственность`

Основа:
`Игумнов`, глава 4.

Уроки:

1. Охрана труда и допуск к работе
2. Инструктажи и обучение
3. СИЗ и безопасное поведение
4. Аварии, инциденты, сообщение об опасности
5. Личная ответственность стропальщика

Интерактив:

- схемы;
- чек-листы;
- тесты на различение обязанностей и запретов.

## Что готовим в первую очередь

Приоритет на `2026-06-16`:

1. Полный пакет `Модуль 1`
2. Стартовую презентацию для `Модуль 1`
3. Банк вопросов для мини-теста
4. Реестр медиа и схем

## Готовность модуля к сборке в iSpring

Модуль можно считать готовым к сборке, когда для него есть:

- список уроков;
- экранные тезисы;
- перечень визуалов;
- список типовых ошибок;
- вопросы мини-теста;
- стартовый `.pptx`.

## Следующий производственный цикл

После сборки `Модуля 1` таким же образом готовим:

1. пакет `Модуль 2`;
2. пакет `Модуль 3`;
3. пакет `Модуль 4`;
4. итоговый тест;
5. финальную сборку курса в iSpring.
