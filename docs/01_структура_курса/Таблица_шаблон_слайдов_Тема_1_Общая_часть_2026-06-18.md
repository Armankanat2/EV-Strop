## Как использовать таблицу

1. Сначала собираем все основные слайды `S001-S005`.
2. Затем добавляем только те подвальные слайды, которые реально нужны для понимания.
3. Контур `5-6 разряд` в `Теме 1` открывается через `S005-P56-01`.
4. Если в сборке решим упростить вводную часть, опциональные `подподвалы` можно не создавать.

## Минимальный обязательный набор для первой сборки

Если нужно быстро собрать рабочий прототип, достаточно сначала сделать:

- `S001`

- `S004`
- `S005`

Остальные слайды можно добавлять вторым проходом как справочные выносы.
