# Утвержденная структура онлайн-презентации на основе пособия Игумнова

Дата утверждения: `2026-06-18`

## Статус

Это утвержденная рабочая схема для `Концепта 1 / Игумнов`.

Ее нужно считать основной опорой для:

- сборки презентации в `PowerPoint`;
- дальнейшей упаковки в `iSpring`;
- подготовки фото, схем, видео и тестов;
- распределения материалов по модулям и промежуточным проверкам.

## Правило навигации

Для этой версии курса утверждена следующая навигационная логика:

1. Основная структура сайта и презентации идет строго линейно, от слайда к слайду.
2. Линейная последовательность должна совпадать с утвержденной схемой курса.
3. Некоторые слайды могут содержать выделенные термины, слова или схемы, которым нужно отдельное пояснение.
4. Такие выделенные элементы должны вести не на следующий слайд курса, а на отдельный поясняющий слайд.
5. Поясняющий слайд не входит в основную линейную цепочку курса и считается вынесенным в `подвал`.
6. На каждом таком поясняющем слайде должна быть обратная ссылка на тот линейный слайд, с которого пользователь ушел в пояснение.
7. После возврата пользователь должен продолжать курс с того же места основной последовательности.

Дополнительное правило для курса:

8. Для тем, где нужен материал повышенного уровня, на основном слайде можно размещать крупную плашку `Стропальщик 5-6 разряд`.
9. Такая плашка должна вести в отдельный подвальный слайд уровня `5-6 разряд`.
10. Слайды уровня `5-6 разряд` должны визуально отличаться от основного курса.
11. Слайды уровня `5-6 разряд` могут вести в `подподвальные` слайды второго уровня вложенности.
12. На `подподвальных` слайдах также должен быть предусмотрен обратный переход.

## Общая логика презентации

1. `Тема 1. Общая часть`

3. `Тема 3. Охрана труда`
4. `Тема 4. Итоговое тестирование`

## Тема 1. `Общая часть`

Назначение блока:
коротко ввести слушателя в тему и задать рамку всего курса.

## Тема 2. `Стропальные работы`

Это основной учебный блок презентации.

Внутри него утверждены три подтемы.

### Тема 2.1. `Строповка грузов`

Состав раздела:

- строповка грузов: описание, картинки, схемы;
- сведения о грузах;
- правила строповки.

Предпочтительный формат подачи:

- по сведениям о грузах: картинки;
- по правилам строповки: лучше использовать видео;
- по базовым объяснениям: короткое текстовое сопровождение и схемы.

Контроль:

- промежуточный тест.

### Тема 2.2. `Технология работ`

Состав раздела:

- обязанности стропальщика;
- сигнализация при перемещении грузов кранами;
- складирование грузов.

Предпочтительный формат подачи:

- по сигнализации: картинки и видео;
- по складированию: схемы и картинки;
- по обязанностям: короткие алгоритмы и пошаговые экраны.

Контроль:

- промежуточный тест.

### Тема 2.3. `Производство работ`

Состав раздела:

- опасная зона;
- требования безопасности при производстве работ;
- производство работ стреловыми кранами вблизи ЛЭП.

Предпочтительный формат подачи:

- по опасной зоне: схемы и картинки;
- по требованиям безопасности: картинки, запреты, визуальные примеры;
- по работам вблизи ЛЭП: картинки и разбор опасных ситуаций.

Контроль:

- промежуточный тест.

## Тема 3. `Охрана труда`

Состав раздела:

- охрана труда: общие сведения;
- охрана труда на рабочем месте стропальщика;
- частные случаи из практики: нарушения.

Предпочтительный формат подачи:

- по общим сведениям и рабочему месту: короткие тезисы, схемы, памятки;
- по частным случаям нарушений: лучше использовать видео.

Контроль:

- промежуточный тест.

## Тема 4. `Итоговое тестирование`

Формат итогового блока:

- итоговое тестирование;
- квалификационный экзамен.

## Производственная трактовка для сборки

Для внутренней сборки в проекте эту утвержденную схему удобно читать так:

1. `Тема 1. Общая часть` = вводный блок курса.
2. `Тема 2.1. Строповка грузов` = `Модуль 1`.
3. `Тема 2.2. Технология работ` = `Модуль 2`.
4. `Тема 2.3. Производство работ` = `Модуль 3`.
5. `Тема 3. Охрана труда` = `Модуль 4`.
6. После `Темы 2.1`, `Темы 2.2`, `Темы 2.3` и `Темы 3` нужен отдельный промежуточный тест.
7. В конце курса идет `Тема 4. Итоговое тестирование` как отдельный блок `Квалификационный экзамен`.

## Как трактовать `подвал`

Под `подвалом` в проекте понимаем отдельный набор вынесенных поясняющих слайдов, которые:

- не участвуют в линейной нумерации основного маршрута;
- открываются только по ссылке с основного слайда;
- содержат расшифровку термина, слова, схемы или дополнительного пояснения;
- обязательно содержат кнопку или ссылку `Вернуться`, ведущую обратно в исходный слайд курса.

Отдельный частный случай `подвала`:

- специальные слайды `Стропальщик 5-6 разряд`, открываемые по крупной плашке с основного слайда.

## Как трактовать `подподвал`

Под `подподвалом` в проекте понимаем второй уровень вынесенных слайдов, который открывается уже из подвального слайда.

Используем его только тогда, когда материала на одном слайде `5-6 разряд` недостаточно и нужно отдельное пояснение более глубокого уровня.
