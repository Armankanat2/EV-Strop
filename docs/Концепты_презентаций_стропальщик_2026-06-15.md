# Концепты презентаций по проекту `ПРОЕКТ ЭВ СТРОПАЛЬЩИК`

Дата фиксации: `2026-06-15`

## Решение

В проекте готовятся не одна универсальная презентация, а два отдельных концепта:

1. `Концепт 1` на основании пособия Игумнова.
2. `Концепт 2` на основании типовой инструкции стропальщика.

Это решение нужно считать рабочей рамкой для всех следующих презентационных материалов.

## Концепт 1. На основании пособия Игумнова

Основной источник:
`Игумнов С. Г. Стропальщик. Производство стропальных работ`

Рабочий текстовый источник для разработки:
`E:\ДПО презентации\ДОП материалы для проектов\СТРОПАЛЬЩИК\Пособие\igumnov-2012-stropalnye-raboty-book.canonical.en.md`

Резервная машинная копия для сверки:
`E:\ДПО презентации\ДОП материалы для проектов\СТРОПАЛЬЩИК\Пособие\igumnov-2012-stropalnye-raboty-book.en.md`

Роль концепта:

- дать учебную и тематическую логику;
- собрать материал как последовательное объяснение профессии;
- выстроить тему от грузов и строповки к технологии работ, опасным зонам и охране труда.

Главная логика:

`грузы -> строповка -> обязанности -> перемещение -> складирование -> опасные зоны -> безопасность`

Что хорошо решает этот концепт:

- базовую структуру учебной презентации;
- понятную подачу для первичного изучения;
- поэтапное объяснение тем;
- привязку к разрядам и к типам грузов.

Актуальная утвержденная схема на `2026-06-18`:

- `Общая часть`;
- `Стропальные работы`, внутри:
  `Строповка грузов` -> `Технология работ` -> `Производство работ` -> `Охрана труда`;
- промежуточный тест после каждого из четырех разделов;
- `Итоговое тестирование (Квалификационный экзамен)` в финале.

Рабочая фиксация утвержденной версии:

- `Утвержденная_структура_онлайн_презентации_Игумнов_2026-06-18.md`

Какие документы уже относятся к этому концепту:

- `Структура_презентации_на_основе_пособия_2-4_разряд.md`
- `Структура_презентации_на_основе_пособия_5-6_разряд.md`
- `Основа_по_пособию_Игумнов_подробный_каркас_курса.md`

Рабочее правило:
для `Концепта 1` в первую очередь используем подготовленный `canonical md`, а не исходную папку пособия и не печатную копию как основной рабочий текст.

## Концепт 2. На основании типовой инструкции

Основной источник:
`Типовая инструкция для стропальщиков по безопасному производству работ грузоподъемными машинами (РД 10-107-96)`

Роль концепта:

- дать рабочую, нормативную и практическую логику;
- строить презентацию не по теоретическим темам, а по этапам работы стропальщика;
- усилить блоки обязанностей, проверок, запретов, аварийных действий и ответственности.

Главная логика:

`допуск -> подготовка -> строповка -> подъем и перемещение -> опускание -> аварийные действия -> ответственность`

Что хорошо решает этот концепт:

- презентацию по обязательным действиям стропальщика;
- практико-ориентированную подачу;
- чек-листы, алгоритмы и запрещенные действия;
- контроль знаний по безопасному поведению.

Текущая опора для разработки:

- `Струкура_на_базе_типовой_инструкции.md`

## Главное различие между концептами

`Концепт 1` отвечает на вопрос:
`что должен понимать стропальщик по теме и как устроена предметная область`.

`Концепт 2` отвечает на вопрос:
`что именно должен делать, проверять и не допускать стропальщик по ходу работы`.

## Как использовать концепты вместе

- не смешивать их в один безликий вариант;
- сначала понимать, какой именно результат мы собираем: учебный или нормативно-практический;
- использовать `Концепт 1` как более объясняющий и учебный;
- использовать `Концепт 2` как более жесткий, регламентный и прикладной;
- при необходимости брать из второго концепта обязательные запреты и контрольные точки даже для презентации первого концепта.

## Рабочее правило на следующий этап
