# Документы проекта

Сюда складываются:

- ТЗ, описания и рабочие постановки;
- готовые материалы для передачи;
- промежуточные сводки, которые важно хранить как документы проекта.

Если документ временный или черновой, лучше сначала держать его в `notes/` или `temp/`.

## Уже зафиксированные опорные документы

- `source-basis.md` — правила использования основных источников проекта;
- `Реестр_обработанных_пособий_2026-06-14.md` — актуальный список обработанных книжных источников и рабочих версий;
- `Единая_карта_тем_и_модулей_стропальщик_2026-06-15.md` — сводная рабочая схема модулей, источников и обязательного контура;
- `Концепты_презентаций_стропальщик_2026-06-15.md` — фиксация двух отдельных презентационных концептов проекта;
- `Утвержденная_структура_онлайн_презентации_Игумнов_2026-06-18.md` — утвержденная схема онлайн-презентации для `Концепта 1 / Игумнов`;
- `Надстройка_5-6_разряд_в_онлайн_курсе_Игумнов_2026-06-18.md` — правило встраивания дополнительного контура для `5-6 разрядов` поверх основного курса `2-4`;
- `Схема_нумерации_слайдов_основной_подвал_подподвал_2026-06-18.md` — единый стандарт нумерации для линейных, подвальных и подподвальных слайдов;
- `Таблица_шаблон_слайдов_Часть_1_Общая_часть_2026-06-18.md` — готовая таблица первой части курса в принятой нумерации;
- `Пакет_наполнения_Модуль_1_Строповка_грузов_2026-06-16.md` — первый производственный пакет для сборки курса;
- `Банк_вопросов_Модуль_1_Строповка_грузов_2026-06-16.md` — стартовый тестовый банк по модулю 1;
- `Пакет_наполнения_Модуль_2_Технология_работы_стропальщика_2026-06-18.md` — производственный пакет для следующего модуля курса;
- `Банк_вопросов_Модуль_2_Технология_работы_стропальщика_2026-06-18.md` — стартовый тестовый банк по модулю 2;
- тематические `.md/.docx/.pdf` в этой папке — производные рабочие материалы по НПА, кейсам, структурам и блокам презентаций.
