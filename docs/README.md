# Документы проекта

Сюда складываются:

- ТЗ, описания и рабочие постановки;
- готовые материалы для передачи;
- промежуточные сводки, которые важно хранить как документы проекта.

Если документ временный или черновой, лучше сначала держать его в `notes/` или `temp/`.

## Уже зафиксированные опорные документы

- `source-basis.md` — правила использования основных источников проекта;
- `Реестр_обработанных_пособий_2026-06-14.md` — актуальный список обработанных книжных источников и рабочих версий;
- `Единая_карта_тем_и_модулей_стропальщик_2026-06-15.md` — сводная рабочая схема модулей, источников и обязательного контура;
- `Концепты_презентаций_стропальщик_2026-06-15.md` — фиксация двух отдельных презентационных концептов проекта;
- тематические `.md/.docx/.pdf` в этой папке — производные рабочие материалы по НПА, кейсам, структурам и блокам презентаций.
