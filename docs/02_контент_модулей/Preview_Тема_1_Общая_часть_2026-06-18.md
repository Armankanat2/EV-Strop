## `S001` Титульный

**Что видит пользователь на экране**

- ЭВ Стропальщик
- Стропальщик `2-4 разряда`
- Стропальщик `5-6 разряда`
- Практический курс по безопасной работе с грузами
- Базовая часть курса - для `2-4` разряда. Для `5-6` разряда открывайте дополнительные блоки.

**Как выглядит**

- крупное титульное фото: стропальщик, кран, груз;
- спокойный производственный фон;
- кнопка `Начать курс`.

**Переход**

- `S001` -> `S002`

## `S002` Видеоинструкция по курсу и глосарий

**Что видит пользователь на экране**

- `Видеоинструкция` - это навигация по курсу, откройте и ознакомьтесь
- `Словарь терминов` - откройте и прочтите

**Как выглядит**

- экран из двух крупных зон или карточек;
- слева можно разместить превью видео с кнопкой запуска;
- справа компактный вход в словарь терминов с иконкой и примерами терминов: `строп`, `строповка`, `груз`, `СИЗ`.

**Что объясняет видео**

- курс проходится по порядку, шаг за шагом;
- в курсе есть основные темы, подробные блоки и тесты;
- по кнопкам можно открывать дополнительные пояснения и потом возвращаться обратно;
- в конце будет итоговый тест;
- блок `5-6 разряд` открывают только те, кому он нужен.

**Переходы**

- `Видеоинструкция` -> `S002-P01`
- `Словарь терминов` -> `S002-P02`
- `S002` -> `S003`

## `S004` Кто такой стропальщик

**Что видит пользователь на экране**

- Стропальщик:
- Принимает и оценивает груз
- Выбирает строп и способ строповки
- Подает сигналы при перемещении
- Следит за безопасностью работы

**Как выглядит**

- фото рабочего процесса;
- простая схема:
  `груз -> строповка -> сигнал -> перемещение -> установка`

**Переходы**

- `S004` -> `S005`
- `Что делает стропальщик по этапам` -> `S004-P01`

## `S005` Кто может работать стропальщиком

**Что видит пользователь на экране**

- Может работать стропальщиком тот, кто:
- Прошел обучение
- Прошел инструктаж
- Прошел проверку знаний
- Допущен к работе
- Использует СИЗ

**Как выглядит**

- чек-лист допуска;
- иконки `обучение / инструктаж / проверка / допуск / СИЗ`.

**Переходы**

- `S005` -> `S006`
- `Что такое инструктаж` -> `S005-P01`
- `Что такое проверка знаний` -> `S005-P02`
- `Что входит в СИЗ` -> `S005-P03`

## `S006` Разряды `2-4`

**Что видит пользователь на экране**

- Разряды, которые закрывает курс:
- `2 разряд` — простые грузы до `5 т`
- `3 разряд` — простые грузы свыше `5 до 25 т`
- `4 разряд` — сложные грузы до `5 т`

Нижняя подпись:

- Этот курс дает базу для работы стропальщика `2-4 разряда`.

**Как выглядит**

- короткая таблица из `3` строк;
- цветовое деление по разрядам;
- крупная плашка `Стропальщик 5-6 разряд`.

**Переходы**

- `S006` -> `S007`
- `Как понимать простые и сложные грузы` -> `S006-P01`
- `Стропальщик 5-6 разряд` -> `S006-P56-01`
