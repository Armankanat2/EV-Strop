# Озвучка `S002-P01`: две версии

Дата фиксации: `2026-06-20`

## Назначение

Это отдельный рабочий файл с двумя вариантами озвучки для `S002-P01`:

- полная дикторская версия с паузами;
- короткая версия примерно на `45-60 секунд`.

## Версия 1. Полная дикторская с паузами

```text
Перед вами видеоинструкция по курсу «Стропальщик».

В этом ролике / коротко объясним, / как проходить курс / и как с ним работать.

Курс интерактивный.
Это значит, / что вы не просто смотрите материал подряд.
Вы будете переходить по слайдам, / открывать дополнительные пояснения / и отвечать на вопросы.

Курс нужно проходить по порядку.
Сначала идет Тема 1.
Потом / Тема 2.
Потом / Тема 3.
После этого / идет итоговое тестирование.

Некоторые темы / разбиты на более подробные блоки.
Их тоже / нужно проходить по порядку.

После тем / и отдельных блоков / будут тесты для самопроверки.

Если на слайде / есть активная кнопка, / нажмите на нее.
Так вы откроете / дополнительную информацию.
Это может быть / термин, / картинка, / схема / или пояснение.

После просмотра / нажмите кнопку / «НАЗАД К КУРСУ».
И продолжайте обучение.

Если в промежуточном тесте / ответ неверный, / откроется ссылка на пояснение.
Откройте ее.
Посмотрите материал.
Потом / вернитесь к вопросу / и продолжайте.

В конце курса / будет итоговый тест.
В нем / 20 вопросов.
Допускается / не больше 3 ошибок.

В итоговом тесте / подсказок уже не будет.
Если тест не сдан, / его можно пройти еще раз.
При повторной попытке / вопросы будут уже другими.

Этот курс / дает базу / для стропальщика 2–4 разряда.

Если вы проходите подготовку / на 5–6 разряд, / на некоторых слайдах / увидите плашку:
«5–6 разряд — пройдите обучение».

Если такая плашка есть, / откройте ее / и изучите дополнительный материал.

Для слушателя / 2–4 разряда / этот дополнительный блок / не обязателен.

На этом все.
Желаем вам / спокойной / и понятной работы с курсом.
```

## Как использовать

1. Для `S002-P01` рабочей основной версией сейчас считать короткую запись `S002-P01_ozvuchka_korotkaya_2026-06-20.mp3`.
2. Полную версию держать как запасную, если позже понадобится более подробный ролик.
