# Презентации

Папка для рабочих и итоговых презентационных материалов.

На `2026-06-15` в проекте зафиксированы два отдельных презентационных концепта:

1. `Концепт 1 / Игумнов` — учебно-тематическая логика на базе пособия.
2. `Концепт 2 / Типовая инструкция` — нормативно-практическая логика на базе типовой инструкции.

На `2026-06-18` для `Концепта 1 / Игумнов` отдельно утверждена рабочая схема онлайн-презентации:
`docs/01_структура_курса/Утвержденная_структура_онлайн_презентации_Игумнов_2026-06-18.md`

- `generated/` - автоматически или полуавтоматически собранные версии;
- `archive/` - предыдущие сохраненные версии;
- `drafts/` - черновые или вспомогательные заготовки презентаций, которые еще не вошли в основной production-контур;
- корень `presentations/` - актуальные рабочие файлы, если нужно держать их под рукой.

Если итог проекта окажется не презентационным, папку можно использовать под любые сборки финальных материалов или оставить резервной.
