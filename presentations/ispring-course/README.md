# iSpring Course Workspace

Рабочая зона для сборки курса `ЭВ Стропальщик` в iSpring.

## Что лежит здесь

- `module-01-stropovka-gruzov` - первый производственный пакет для сборки модуля.
- стартовые `.pptx`-файлы, если они собираются отдельно от старых концептов.
- вспомогательные сценарии слайдов и планы интерактивов.

## Как использовать

1. Открыть общий каркас курса в `docs/Каркас_iSpring_курса_ЭВ_Стропальщик_2026-06-16.md`.
2. Работать по пакету текущего модуля.
3. Собирать слайды в PowerPoint.
4. Добавлять интерактивы и тесты в iSpring.
5. Параллельно пополнять `assets/course-media`.

## Текущий фокус

Первым в сборку идет `Модуль 1. Строповка грузов`.
