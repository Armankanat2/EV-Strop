# iSpring Course Workspace

Рабочая зона для сборки курса `ЭВ Стропальщик` в iSpring.

## Что лежит здесь

- `module-01-stropovka-gruzov` - первый производственный пакет для сборки модуля.
- `module-02-tekhnologiya-raboty-stropalshchika` - второй производственный пакет для следующего модуля.
- стартовые `.pptx`-файлы, если они собираются отдельно от старых концептов.
- вспомогательные сценарии слайдов и планы интерактивов.

## Как использовать

1. Открыть общий каркас курса в `docs/Каркас_iSpring_курса_ЭВ_Стропальщик_2026-06-16.md`.
2. Работать по пакету текущего модуля.
3. Собирать слайды в PowerPoint.
4. Добавлять интерактивы и тесты в iSpring.
5. Параллельно пополнять `assets/course-media`.

## Правило навигации

- основной курс собирается в линейной последовательности от слайда к слайду;
- дополнительные пояснения по терминам, словам и схемам выносятся в отдельные справочные слайды;
- такие справочные слайды считаются `подвалом` и не входят в основную линейную цепочку;
- на каждом выносном слайде должна быть кнопка возврата на исходный слайд.
- для материала `5-6 разряд` на основном слайде можно ставить крупную плашку перехода;
- такой переход ведет на отдельный подвальный слайд `5-6 разряд` с отличающимся дизайном;
- из такого слайда допускается переход в `подподвал`;
- в `подподвале` обязательно должен быть возврат в слайд `5-6 разряд`.

Для идентификаторов и ссылок используем стандарт:
`docs/Схема_нумерации_слайдов_основной_подвал_подподвал_2026-06-18.md`
